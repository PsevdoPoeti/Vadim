<p class="tab">
<font size="6" color="black">
<b>
Тобиас Раушер
</b>
, гитарист в жанре Melodic Percussive Fingerstyle. Рождён в столице Германии.В течение долгих лет работал в качестве частного преподавателя гитары. В 2008 году начал изучение музыки в одном из самых старых городов Германии, Кельне, и имел несколько проектов с акустической гитарой.В 2010 году полностью перешёл в игру на акустической гитаре жанра (ударного) фингерстайл. Тобис создавал свои экспериментальные строи, которые служили для создания новых интересных идей (композиций), и подчёркивали его творческую натуру. Дня берлинского музыканта, как первопроходца в новой схеме этого жанра, открывались большие возможности, позволяющие создать что-то, действительно, новое.
